# MyConsoleApp

## Overview
MyConsoleApp is a simple C# console application built with .NET.  
It demonstrates basic input/output operations and serves as a starting point for learning or extending into larger projects.

## Features
- Reads user input from the console
- Performs basic operations
- Displays results back to the user
- Organized with `Program.cs` as the entry point

## Requirements
- .NET 6.0 SDK or later
- Visual Studio or Visual Studio Code
